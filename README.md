# Projeto 1 Computação Visual - Equalização de Histograma

**Professor:** André Kishimoto

- Pietro Zanaga Neto (RA 10418574)

## Contexto do Projeto

Este projeto consiste na implementação de um programa em C utilizando a biblioteca SDL3 
para carregar uma imagem, convertê-la para a escala de cinzas, calcular o histograma de 
intensidade, informar a média de intensidade, o desvio padrão e realizar a equalização do 
histograma, tudo através de uma interface gráfica interativa.
